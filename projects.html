Get Cooking! – VR Cooking Game

Unity · VR interaction · User study design

A VR environment for everyday cooking tasks such as cracking, whisking, and flipping eggs. Designed to support controlled studies on motion, input techniques, and user performance.

Project page · Source code

VR Cybersickness Camera Rig

Unity · Experimental platform

A configurable camera rig that can apply different perturbation patterns (e.g., oscillations, sudden rotations, variable speeds) to systematically study the onset and progression of cybersickness.

EEG-Based Motor Imagery Classification

Python · PyTorch · EEG

Implemented deep learning pipelines, including transformer-based models, to classify motor imagery states from EEG data as part of a broader BCI research effort.
